# emacs-brave

Modern emacs distribution.

## Emacs
- Cocoa Emacs 24.2.1

## Install
```
$ git clone https://github.com/ongaeshi/emacs-brave.git
$ cd emacs-brave
$ emacs -Q -l set-user-emacs-directory.el -l init.el
```

- anzu
- init-loader
